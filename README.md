`lunaris-forge/pipelines` are a series of steps that HTTP requests or responses must pass through.

## Installation

To install the LunarisForge/Pipelines library, use Composer:

```sh
composer require lunaris-forge/pipelines
```

## Testing

Run the tests:

   ```sh
   vendor/bin/phpunit
   ```
